    **Helper: formatRelativeTime(isoString: string): string** — returns relative time like "2h ago", "3d ago". Use `Date.now() - new Date(isoString).getTime()` to compute delta in ms, then format as: <1min="just now", <60min="Xm ago", <24h="Xh ago", <30d="Xd ago", else toLocaleDateString('pt-BR').

    Build AuditQueryParams from state:
    - `limit: pageSize`
    - `offset: currentPage * pageSize`
    - `org_id: selectedOrgId || undefined`

    - `action: selectedActions.length === 1 ? selectedActions[0] : undefined` (API supports single action filter only — if multiple selected, filter client-side)
    - `resource_type: selectedResourceType || undefined`
    - `date_from: dateFrom || undefined`
    - `date_to: dateTo || undefined`
    Call queryAuditLogs(params), set logs/total, handle error with try/catch.
    When multiple actions are selected, apply client-side filter: `response.data.filter(log => selectedActions.includes(log.action))`.
