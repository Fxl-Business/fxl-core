Create supabase/functions/audit-logs/index.ts following the exact same patterns as admin-tenants/index.ts for auth, CORS, and response helpers.

Structure:
1. Imports: serve from deno std, createClient from esm.sh supabase-js
2. Env vars: SUPABASE_URL, SUPABASE_SERVICE_ROLE_KEY (use service role to bypass RLS for reading)
3. corsHeaders: same as admin-tenants
4. CORS preflight handler (OPTIONS -> 204)
5. JWT decode + super_admin validation (same pattern as admin-tenants lines 55-79)
6. Only allow GET method, return 405 for others
7. Parse query params from URL:
   - limit: number, default 50, max 100, min 1 — clamp with Math.min(Math.max(...), ...)
   - offset: number, default 0, min 0
   - org_id: string (optional filter)
   - actor_id: string (optional filter)
   - action: string (optional filter — e.g. 'create', 'update')
   - resource_type: string (optional filter — e.g. 'tenant', 'task')
   - date_from: ISO string (optional — created_at >= date_from)
   - date_to: ISO string (optional — created_at <= date_to)
8. Build Supabase query:
   - Start with supabase.from('audit_logs').select('*', { count: 'exact' })
   - Apply .order('created_at', { ascending: false })
   - Apply each filter conditionally with .eq() when param is present
   - Apply date_from with .gte('created_at', date_from) when present
   - Apply date_to with .lte('created_at', date_to) when present
   - Apply .range(offset, offset + limit - 1) for pagination
9. Handle query error -> 500
10. Return JSON: { data: rows, total: count, limit, offset }

Use getSupabaseAdmin() helper (same pattern as admin-tenants) with service_role key to bypass RLS — the super_admin check at JWT level is the access control.

IMPORTANT per CLAUDE.md:
- NO sub-paths — all params via query string (already planned)
- NO `any` type — use Record<string, unknown> for JWT payload
- Edge functions run in Deno — cannot import from src/, must use inline types or deno-compatible imports
- Add @ts-nocheck comment at top (same as admin-tenants) since Deno runtime is not checked by project tsc
