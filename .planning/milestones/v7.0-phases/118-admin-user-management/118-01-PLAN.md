## Objective

Add a segmented filter control (Todos/Sem org/Com org) to `/admin/users` and an "assign to org" action button on unaffiliated user rows that opens a Dialog for org selection.

<action>
Modify `UsersPage.tsx` to add client-side filtering:

1. Add a `filter` state variable with type `'all' | 'unaffiliated' | 'affiliated'`, defaulting to `'all'`.

2. Compute filtered users and counts from the `users` array:
   ```typescript
   const unaffiliatedUsers = users.filter(u => u.organizationMemberships.length === 0)
   const affiliatedUsers = users.filter(u => u.organizationMemberships.length > 0)
   const filteredUsers = filter === 'all' ? users : filter === 'unaffiliated' ? unaffiliatedUsers : affiliatedUsers
   ```

3. Compute counts for each segment:
   ```typescript
   const counts = { all: users.length, unaffiliated: unaffiliatedUsers.length, affiliated: affiliatedUsers.length }
   ```

4. Replace the existing stats bar with a segmented control using three Button components styled as a button group. Each button:
   - Label: "Todos ({count})", "Sem org ({count})", "Com org ({count})"
   - Active state: `bg-indigo-600 text-white` (or `bg-indigo-500` in dark mode)
   - Inactive state: `bg-white text-slate-600 hover:bg-slate-50` (dark: `bg-card text-slate-400 hover:bg-slate-800`)
   - Wrap all three in a `div` with `inline-flex rounded-lg border border-slate-200 dark:border-slate-700 p-0.5 gap-0.5` for the group container

5. Use `filteredUsers` instead of `users` in the user list rendering loop and empty state check.

6. Update the total count display to show `filteredUsers.length` of `totalCount`.
</action>

<action>
Extend `UsersPage.tsx` with org assignment functionality:

1. Add imports:
   ```typescript
   import { Link2 } from 'lucide-react'
   import { Dialog, DialogContent, DialogHeader, DialogTitle, DialogDescription, DialogFooter } from '@shared/ui/dialog'
   import { Select, SelectContent, SelectItem, SelectTrigger, SelectValue } from '@shared/ui/select'
   import { listTenants, setClerkTokenGetter } from '@platform/services/tenant-service'
   import { addOrgMember } from '@platform/services/admin-service'
   import { toast } from 'sonner'
   import type { Tenant } from '@platform/types/tenant'
   ```

2. Add state variables:
   ```typescript
   const [assignDialogOpen, setAssignDialogOpen] = useState(false)
   const [assigningUser, setAssigningUser] = useState<AdminUser | null>(null)
   const [selectedOrgId, setSelectedOrgId] = useState('')
   const [tenants, setTenants] = useState<Tenant[]>([])
   const [tenantsLoading, setTenantsLoading] = useState(false)
   const [assignLoading, setAssignLoading] = useState(false)
   ```

3. Register tenant-service token getter in the existing `useEffect` that registers admin token:
   ```typescript
   useEffect(() => {
     if (session) {
       setAdminClerkTokenGetter(() => session.getToken({ template: 'supabase' }))
       setClerkTokenGetter(() => session.getToken({ template: 'supabase' }))
     }
   }, [session])
   ```

4. Add `fetchTenants` function:
   ```typescript
   async function fetchTenants() {
     setTenantsLoading(true)
     try {
       const result = await listTenants()
       setTenants(result.tenants ?? [])
     } catch (err) {
       console.error('Failed to fetch tenants:', err)
       setTenants([])
     } finally {
       setTenantsLoading(false)
     }
   }
   ```

5. Add `openAssignDialog` function that sets `assigningUser`, opens dialog, and calls `fetchTenants()`.

6. Add `handleAssign` function:
   ```typescript
   async function handleAssign() {
     if (!assigningUser || !selectedOrgId) return
     setAssignLoading(true)
     try {
       await addOrgMember(selectedOrgId, assigningUser.id)
       toast.success('Usuario vinculado com sucesso')
       setAssignDialogOpen(false)
       setAssigningUser(null)
       setSelectedOrgId('')
       await fetchUsers()
     } catch (err) {
       toast.error(err instanceof Error ? err.message : 'Erro ao vincular usuario')
     } finally {
       setAssignLoading(false)
     }
   }
   ```

7. In the user list row, after the org badges section (the `<div className="hidden shrink-0 ...">` block), add a "Vincular" button for unaffiliated users only:
   ```tsx
   {user.organizationMemberships.length === 0 && (
     <Button
       variant="outline"
       size="sm"
       className="shrink-0 gap-1.5 text-xs"
       onClick={() => openAssignDialog(user)}
     >
       <Link2 className="h-3 w-3" />
       Vincular
     </Button>
   )}
   ```

8. Add the Dialog component at the end of the JSX return (inside the outer div), with:
   - DialogHeader showing user name and email
   - Select dropdown with all tenants as options (SelectItem value={tenant.id}, label={tenant.name})
   - Loading state for tenants (tenantsLoading)
   - DialogFooter with Cancel and "Vincular" buttons
   - "Vincular" button disabled when `!selectedOrgId || assignLoading`

9. Reset `selectedOrgId` to `''` when dialog closes via `onOpenChange`.
</action>
