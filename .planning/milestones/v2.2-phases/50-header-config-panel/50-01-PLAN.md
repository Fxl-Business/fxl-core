Create the HeaderConfigPanel component — a Sheet-based panel with grouped form controls for all header configuration options. This component receives the current header config and an update callback, and renders Switch toggles, an Input for brandLabel, and a Select for periodType.

**Create `tools/wireframe-builder/components/editor/HeaderConfigPanel.tsx`**

Props interface:
```typescript
import type { HeaderConfig, PeriodType } from '@tools/wireframe-builder/types/blueprint'

type Props = {
  open: boolean
  headerConfig: HeaderConfig
  configLabel: string  // fallback label from config.label (shown as placeholder)
  onUpdate: (updater: (header: HeaderConfig) => HeaderConfig) => void
  onClose: () => void
}
```

The `onUpdate` callback receives an updater function that transforms the current HeaderConfig. This pattern matches `updateWorkingConfig` — the WireframeViewer will wrap it to call `updateWorkingConfig((cfg) => ({ ...cfg, header: updater(cfg.header ?? {}) }))`.

**Component structure:**

```tsx
export default function HeaderConfigPanel({
  open,
  headerConfig,
  configLabel,
  onUpdate,
  onClose,
}: Props) {
  return (
    <Sheet open={open} onOpenChange={(v) => !v && onClose()}>
      <SheetContent side="right" className="w-[400px] sm:w-[450px] overflow-y-auto">
        <SheetHeader>
          <SheetTitle>Header Config</SheetTitle>
          <SheetDescription>Configure a aparencia do header do dashboard</SheetDescription>
        </SheetHeader>

        <div className="mt-6 space-y-6">
          {/* Group 1: Aparencia */}
          {/* Group 2: Periodo */}
          {/* Group 3: Usuario */}
          {/* Group 4: Acoes */}
        </div>
      </SheetContent>
    </Sheet>
  )
}
```

**Group 1: Aparencia**

Section heading: "Aparencia"
- **Exibir Logo** — Switch toggle for `showLogo` (default: true when undefined)
  ```tsx
  <div className="flex items-center justify-between">
    <div>
      <Label htmlFor="showLogo">Exibir Logo</Label>
      <p className="text-xs text-muted-foreground">Mostra o logo ou nome da marca no header</p>
    </div>
    <Switch
      id="showLogo"
      checked={headerConfig.showLogo !== false}
      onCheckedChange={(checked) =>
        onUpdate((h) => ({ ...h, showLogo: checked }))
      }
    />
  </div>
  ```

- **Label Customizado** — Input for `brandLabel`
  ```tsx
  <div className="space-y-2">
    <Label htmlFor="brandLabel">Label Customizado</Label>
    <Input
      id="brandLabel"
      placeholder={configLabel}
      value={headerConfig.brandLabel ?? ''}
      onChange={(e) =>
        onUpdate((h) => ({
          ...h,
          brandLabel: e.target.value || undefined,
        }))
      }
    />
    <p className="text-xs text-muted-foreground">
      Substitui o nome padrao "{configLabel}" no header
    </p>
  </div>
  ```
  When the input is empty, set `brandLabel` to `undefined` so it falls back to `config.label`.

**Group 2: Periodo**


  ```tsx
    <div className="space-y-2">
      <Label htmlFor="periodType">Tipo de Periodo</Label>
      <Select
        value={headerConfig.periodType ?? 'mensal'}
        onValueChange={(value) =>
  ```

```tsx
function ActionToggle({

  description: string
  checked: boolean
  onCheckedChange: (checked: boolean) => void

}
```
