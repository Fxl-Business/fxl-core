    **2. Create `supabase/functions/admin-tenants/index.ts`:**

    Follow the exact same Deno Edge Function pattern as `auth-token-exchange/index.ts` (same CORS setup, same `Deno.serve()` pattern).

    The function handles 3 routes via URL path + HTTP method:
    - `GET /admin-tenants` -> List all Clerk orgs
    - `GET /admin-tenants/:orgId` -> Get single org detail
    - `POST /admin-tenants` -> Create new org

    **Auth check (CRITICAL):**
    - Extract JWT from `Authorization: Bearer <token>` header
    - Decode the JWT (base64 URL decode the payload â€” no need for full verification since Supabase already verified it at the gateway)
    - Check that `payload.super_admin === true`
    - If not super_admin, return 403 `{ error: "Forbidden: super_admin required" }`

    **Environment variables needed:**
    - `CLERK_SECRET_KEY` â€” read via `Deno.env.get('CLERK_SECRET_KEY')`

    **Clerk API calls:**
    - Base URL: `https://api.clerk.com/v1`
    - Auth header: `Authorization: Bearer ${CLERK_SECRET_KEY}`
    - List orgs: `GET /organizations?limit=100&order_by=-created_at`
    - Get org: `GET /organizations/${orgId}`
    - Create org: `POST /organizations` with body `{ name, slug?, created_by: <user_id from JWT> }`

    **Response mapping for list:**
    Map each Clerk org response to `Tenant` shape:
    ```typescript
    {
      id: org.id,
      name: org.name,
      slug: org.slug,
      membersCount: org.members_count,
      createdAt: org.created_at,
      imageUrl: org.image_url,
    }
    ```

    **Response mapping for detail:**
    Same as Tenant plus:
    ```typescript
    {
      ...tenantFields,
      publicMetadata: org.public_metadata,
      privateMetadata: org.private_metadata,
      maxAllowedMemberships: org.max_allowed_memberships,
      adminDeleteEnabled: org.admin_delete_enabled,
    }
    ```

    **CORS headers** (same pattern as auth-token-exchange):
    ```typescript
    const corsHeaders = {
      'Access-Control-Allow-Origin': '*',
      'Access-Control-Allow-Headers': 'authorization, x-client-info, apikey, content-type',
    }
    ```

    Handle OPTIONS preflight requests returning 204 with CORS headers.

    **Error handling:**
    - Missing CLERK_SECRET_KEY -> 500 `{ error: "Server configuration error" }`
    - Clerk API errors -> forward status code and error message
    - Invalid route -> 404 `{ error: "Not found" }`

    ```typescript
    export async function getTenantDetail(orgId: string): Promise<TenantDetail> {
      const res = await fetch(`${FUNCTIONS_URL}/admin-tenants/${orgId}`, {
        headers: await getAuthHeaders(),
      })
      if (!res.ok) throw new Error(`Failed to get tenant: ${res.status}`)
      return res.json()
    }
    ```

    **3. `createTenant(payload: CreateTenantPayload)`:**
    ```typescript
    export async function createTenant(payload: CreateTenantPayload): Promise<TenantDetail> {
      const res = await fetch(`${FUNCTIONS_URL}/admin-tenants`, {
        method: 'POST',
        headers: await getAuthHeaders(),
        body: JSON.stringify(payload),
      })
      if (!res.ok) {
        const err = await res.json().catch(() => ({ error: 'Unknown error' }))
        throw new Error(err.error ?? `Failed to create tenant: ${res.status}`)
      }
      return res.json()
    }
    ```
