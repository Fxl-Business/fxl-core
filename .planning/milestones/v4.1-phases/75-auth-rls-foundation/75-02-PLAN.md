The user must update the Clerk JWT template in the Clerk Dashboard. This cannot be automated — Clerk does not expose a JWT template API.

**Steps for the user:**

1. Go to **Clerk Dashboard** -> **JWT Templates**
2. Find the template named **"supabase"** (or create one if it doesn't exist)
3. Update the template claims to include `super_admin`:

```json
{
  "sub": "{{user.id}}",
  "org_id": "{{org.id}}",
  "role": "{{org_membership.role}}",
  "super_admin": "{{user.public_metadata.super_admin}}"
}
```

4. Save the template

5. Then set `super_admin: true` on the user's publicMetadata:
   - Go to **Clerk Dashboard** -> **Users** -> select your user (Cauet)
   - Click **Edit metadata** (or Public metadata section)
   - Set: `{ "super_admin": true }`
   - Save

6. Log out and log back in to the Nexo app to get a fresh JWT with the new claims
