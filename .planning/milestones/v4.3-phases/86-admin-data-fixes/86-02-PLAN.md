# Plan 86-02: Refactor AdminDashboard to use edge function data

## Objective

Replace the broken `useOrganizationList` Clerk hook in AdminDashboard with calls to the `admin-tenants` and `admin-users` edge functions. Create the service layer and types needed for the admin-users edge function. After this plan, the dashboard "Tenants" metric shows the total count of ALL Clerk organizations and "Usuarios" shows the total count of ALL Clerk users.

- `FUNCTIONS_URL` from `import.meta.env.VITE_SUPABASE_FUNCTIONS_URL ?? ''`

Export one function:

```typescript
import type { AdminUserListResponse } from '@platform/types/admin'

/**
 * List all Clerk users (admin-only).
 * Calls the admin-users Supabase Edge Function.
 */
export async function listUsers(): Promise<AdminUserListResponse> {
  const res = await fetch(`${FUNCTIONS_URL}/admin-users`, {
    headers: await getAuthHeaders(),
  })
  if (!res.ok) throw new Error(`Failed to list users: ${res.status}`)
  return res.json() as Promise<AdminUserListResponse>
}
```

Re-export types for convenience:
```typescript
export type { AdminUser, AdminUserListResponse }
```

**Important:** Do NOT duplicate `setClerkTokenGetter` — the admin-service has its own independent `setAdminClerkTokenGetter`. Components that need both services will register both getters.

```typescript
const { session } = useSession()

const [tenantCount, setTenantCount] = useState(0)
const [userCount, setUserCount] = useState(0)
const [metricsLoading, setMetricsLoading] = useState(true)

// Register token getters for both services
useEffect(() => {
  if (session) {
    setClerkTokenGetter(() => session.getToken({ template: 'supabase' }))
    setAdminClerkTokenGetter(() => session.getToken({ template: 'supabase' }))
  }
}, [session])

// Fetch metrics from edge functions
useEffect(() => {
  if (!session) return

  let cancelled = false

  async function fetchMetrics() {
    setMetricsLoading(true)
    try {
      const [tenantsResult, usersResult] = await Promise.all([
        listTenants(),
        listUsers(),
      ])
      if (!cancelled) {
        setTenantCount(tenantsResult.totalCount)
        setUserCount(usersResult.totalCount)
      }
    } catch (err) {
      console.error('Failed to fetch admin metrics:', err)
      // On error, counts stay at 0 — MetricCard shows 0
    } finally {
      if (!cancelled) setMetricsLoading(false)
    }
  }

  fetchMetrics()
  return () => { cancelled = true }
}, [session])
```

**Update the modules loading logic:**

Change the `useEffect` for `fetchModuleStats` to NOT depend on `isOrgsLoaded` or `userMemberships`. Instead, it should depend on `metricsLoading` and `tenantCount`:

```typescript
useEffect(() => {
  if (metricsLoading || tenantCount === 0) {
    setModulesLoading(false)
    return
  }

  async function fetchModuleStats() {
    // Fetch all tenant_modules rows (no org filter needed — super admin sees all)
    const { data, error } = await supabase
      .from('tenant_modules')
      .select('org_id, module_id, enabled')

    if (error || !data) {
      setAvgModulesPerTenant(totalModuleCount)
      setModulesLoading(false)
      return
    }

    // Collect unique org IDs from tenant_modules
    const orgIds = [...new Set(data.map(row => row.org_id))]
    const denominator = orgIds.length || 1

    // Group by org_id and count enabled modules per tenant
    const perOrg = new Map<string, Set<string>>()
    const disabledPerOrg = new Map<string, Set<string>>()

    for (const row of data) {
      if (row.enabled) {
        if (!perOrg.has(row.org_id)) perOrg.set(row.org_id, new Set())
        perOrg.get(row.org_id)!.add(row.module_id)
      } else {
        if (!disabledPerOrg.has(row.org_id)) disabledPerOrg.set(row.org_id, new Set())
        disabledPerOrg.get(row.org_id)!.add(row.module_id)
      }
    }

    // Calculate active modules per tenant (opt-out model: no row = enabled)
    let totalActive = 0
    for (const orgId of orgIds) {
      const enabledInDb = perOrg.get(orgId)?.size ?? 0
      const disabledInDb = disabledPerOrg.get(orgId)?.size ?? 0
      const modulesInDb = enabledInDb + disabledInDb
      const modulesNotInDb = totalModuleCount - modulesInDb
      totalActive += enabledInDb + modulesNotInDb
    }

    setAvgModulesPerTenant(Math.round((totalActive / denominator) * 10) / 10)
    setModulesLoading(false)
  }

  fetchModuleStats()
}, [metricsLoading, tenantCount, totalModuleCount])
```
