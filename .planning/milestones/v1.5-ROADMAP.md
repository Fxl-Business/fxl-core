# Milestone v1.5: Modular Foundation & Knowledge Base

**Status:** SHIPPED 2026-03-13
**Phases:** 29-33
**Total Plans:** 14

## Overview

Transformou o FXL Core de sistema monolitico em plataforma modular com fronteiras claras entre areas. Introduziu base de conhecimento autoalimentada (bugs, decisoes, padroes, lessons) com full-text search em portugues, gestao de tarefas por cliente com kanban board, e home page como hub operacional com activity feed cross-module.

## Milestone Summary

**Key Decisions:**
- Module manifest pattern: static typed constant in src/modules/registry.ts
- ESLint v9 flat config with boundaries plugin for cross-module import enforcement
- Service layer: each module uses its own lib/[name]-service.ts
- RLS: anon-permissive on new tables, Clerk auth at application layer
- tsvector language: 'portuguese' for KB full-text search
- knowledge_entries column: entry_type (not kind/category)
- Optimistic kanban updates with local useState + refetch rollback on error
- DocumentarButton outside TaskCard — cross-module navigation decoupled

**Requirements:** MOD-01 through MOD-04, KB-01 through KB-07, TASK-01 through TASK-05, HOME-01 through HOME-03 — all 19/19 shipped.
