Wireframe token pattern (from Phase 8):
- All wireframe content uses --wf-* CSS variables via inline styles
- `style={{ color: 'var(--wf-body)' }}` for text
- `style={{ backgroundColor: 'var(--wf-card)', borderColor: 'var(--wf-border)' }}` for cards/containers
- `style={{ color: 'var(--wf-muted)' }}` for secondary text
- `style={{ color: 'var(--wf-accent)' }}` for accents (gold)
- Never use app-level Tailwind tokens (bg-background, text-foreground) in wireframe renderers
