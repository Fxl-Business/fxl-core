**WireframeViewer.tsx** — Add handler and wire:

1. Add handler:
```typescript
function handleAddFilter(filter: FilterOption) {
  updateWorkingScreen((screen) => {
    // Prevent duplicate keys
    if (screen.filters.some((f) => f.key === filter.key)) return screen
    return { ...screen, filters: [...screen.filters, { ...filter }] }
  })
}
```

2. Pass `onAddFilter={handleAddFilter}` to BlueprintRenderer.

3. After adding a filter, automatically select it for editing: call `setSelectedFilterIndex(newIndex)` where newIndex = current filters.length (the newly added filter will be at the end). This gives immediate feedback — the operator adds a preset and can customize it right away.
