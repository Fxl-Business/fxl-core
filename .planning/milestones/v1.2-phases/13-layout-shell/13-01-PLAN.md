**1. Layout.tsx -- Remove nested scroll, enable three-column layout:**
- Remove `overflow-hidden` from the flex parent div
- Remove `flex-col md:flex-row` from the flex parent (just `flex flex-1`)
- Remove `overflow-y-auto` from `<main>`
- Remove the `mx-auto max-w-4xl px-5 py-8 md:px-8 md:py-10` wrapper div around Outlet
- Add `min-w-0 flex-1 px-8 py-10 lg:px-12` to main (min-w-0 prevents flex overflow from wide code blocks)
- Import and render `<ScrollToTop />` as last child inside the outer div
- Keep Sidebar import unchanged

Target Layout.tsx structure:
```tsx
import { Outlet } from 'react-router-dom'
import Sidebar from './Sidebar'
import TopNav from './TopNav'
import ScrollToTop from './ScrollToTop'

export default function Layout() {
  return (
    <div className="flex min-h-screen flex-col bg-background">
      <TopNav />
      <div className="flex flex-1">
        <Sidebar />
        <main className="min-w-0 flex-1 px-8 py-10 lg:px-12">
          <Outlet />
        </main>
      </div>
      <ScrollToTop />
    </div>
  )
}
```

**4. ScrollToTop.tsx -- NEW file:**
```tsx
import { useEffect } from 'react'
import { useLocation } from 'react-router-dom'

export default function ScrollToTop() {
  const { pathname } = useLocation()

  useEffect(() => {
    window.scrollTo(0, 0)
  }, [pathname])

  return null
}
```

- Add heading scroll offset (outside @layer base, after the scrollbar styling block):
```css

}
```
The 5rem (80px) accounts for h-16 header (64px) + 16px buffer. This prepares for Phase 15 TOC scroll-to-heading.
