Gauge needle math (from RESEARCH.md):
```typescript
const RADIAN = Math.PI / 180
// Needle points from center (cx, cy) outward at angle
// value=min → angle=180° (left), value=max → angle=0° (right)
// angleDeg = 180 - ((value - min) / (max - min)) * 180
// SVG: x = cx + r * cos(angleRad), y = cy - r * sin(angleRad)  [SVG Y is inverted]
// Use viewBox="0 0 200 110" for fixed coordinate space (predictable, responsive-safe)
```

    **Zone data computation:**
    When `zones` prop is provided, convert from "upper bound" format to "arc size" format for Pie:
    ```typescript
    const effectiveMin = min ?? 0
    const effectiveMax = max ?? 100
    const total = effectiveMax - effectiveMin

    // If no zones, use default 3-zone split:
    const zoneData = zones
      ? zones.map((z, i) => {
          const prevBound = i === 0 ? effectiveMin : zones[i - 1].value
          return {
            value: z.value - prevBound,   // arc size = upper bound minus previous upper bound
            fill: z.color ?? (i === 0 ? 'var(--wf-negative)' : i === 1 ? '#f59e0b' : 'var(--wf-positive)'),
          }
        })
      : [
          { value: total * 0.4, fill: 'var(--wf-negative)' },
          { value: total * 0.3, fill: '#f59e0b' },
          { value: total * 0.3, fill: 'var(--wf-positive)' },
        ]
    ```

    **Needle SVG approach (absolute positioned overlay):**
    Use a fixed `viewBox="0 0 200 110"` for predictable coordinate math regardless of container width.
    The gauge center in this viewBox: `cx=100, cy=99` (matching cy="90%" of ~110px height ≈ 99px).
    Needle radius: 70 (slightly inside innerRadius which is ~55% of ~82px ≈ 45px → scale up proportionally).

    ```typescript
    const RADIAN = Math.PI / 180
    const effectiveValue = Math.min(effectiveMax, Math.max(effectiveMin, value))
    const pct = (effectiveValue - effectiveMin) / (effectiveMax - effectiveMin)
    const angleDeg = 180 - pct * 180  // 180=left, 0=right
    const angleRad = angleDeg * RADIAN
    // Needle tip in viewBox coords (center at 100, 99)
    const needleTipX = 100 + 70 * Math.cos(angleRad)
    const needleTipY = 99 - 70 * Math.sin(angleRad)  // SVG Y inverted
    ```

    **Container structure:**
    ```tsx
    <div className="rounded-lg border border-wf-card-border bg-wf-card p-4">
      <p className="mb-2 text-sm font-semibold text-wf-heading">{title}</p>
      {/* value display */}
      <p className="mb-1 text-center text-2xl font-bold text-wf-heading">{value}</p>
      {/* gauge with needle overlay */}
      <div className="relative" style={{ height }}>
        <ResponsiveContainer width="100%" height="100%">
          <PieChart>
            <Pie
              data={zoneData}
              startAngle={180}
              endAngle={0}
              innerRadius="55%"
              outerRadius="75%"
              cx="50%"
              cy="90%"
              dataKey="value"
              isAnimationActive={false}
              strokeWidth={0}
            >
              {zoneData.map((zone, i) => (
                <Cell key={i} fill={zone.fill} />
              ))}
            </Pie>
          </PieChart>
        </ResponsiveContainer>
        {/* SVG needle overlay — absolutely positioned over chart */}
        <svg
          className="pointer-events-none absolute inset-0"
          width="100%"
          height="100%"
          viewBox="0 0 200 110"
          preserveAspectRatio="xMidYMid meet"
        >
          {/* needle line */}
          <line
            x1={100}
            y1={99}
            x2={needleTipX}
            y2={needleTipY}
            stroke="var(--wf-heading)"
            strokeWidth={2}
            strokeLinecap="round"
          />
          {/* needle pivot circle */}
          <circle cx={100} cy={99} r={4} fill="var(--wf-heading)" />
        </svg>
      </div>
    </div>
    ```

    Recharts imports: `PieChart, Pie, Cell, ResponsiveContainer` from 'recharts'.
    No `Legend` or `Tooltip` — the gauge is self-explanatory with the value display above.
