<objective>
Restructure the wireframe viewer layout so the header renders full-width above the sidebar, and move the "Gerenciar" button from the sidebar footer to the header.

Purpose: The current layout places sidebar and main content side-by-side (`flex-row`), with the header inside `<main>`. This means the header is only as wide as the content area, not full-width. Standard SaaS dashboard chrome has a top header spanning the full width (over both sidebar and content), with the sidebar below it.

Output: `WireframeHeader.tsx` with optional `onGerenciar` prop. `WireframeViewer.tsx` with `flex-col` outer container, `WireframeHeader` moved above the sidebar+content row, sidebar repositioned with `top: 56px`, and "Gerenciar" button removed from sidebar footer.
</objective>
