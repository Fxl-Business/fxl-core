## Partial Requirements Detail

### MOD-02: Super admin configura modulos por tenant

ModulesPanel at `/admin/modules` fully implements per-tenant module CRUD via Supabase `tenant_modules`. However, TenantDetailPage (`/admin/tenants/:orgId`) still shows a Phase 77 placeholder for the modules section instead of linking to or embedding the module controls. The functionality exists but the navigation path from tenant detail → module management is broken.

**Workaround:** User navigates to `/admin/modules` directly and selects the tenant.

### MCP-02: Clerk MCP server configurado

`.mcp.json` contains configuration for `@anthropic/mcp-server-clerk@latest`, but this npm package does not exist. The config is a placeholder with documentation noting the unavailability. This has no impact on the application itself — it only affects Claude Code's ability to interact with Clerk directly.
