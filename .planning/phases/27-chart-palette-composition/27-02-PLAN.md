    **Types:**
    ```typescript
    type Segment = {
      label: string
      value: number
      color?: string  // defaults to var(--wf-chart-N) by index
    }

    type Props = {
      title: string
      segments: Segment[]
      height?: number        // bar height in px, default 32
      showLegend?: boolean   // default true
      formatValue?: (v: number) => string
    }
    ```

    **Component structure:**
    - Outer container: `rounded-xl border border-wf-card-border bg-wf-card p-4 shadow-sm` (matching v1.4 card aesthetic)
    - Title: `text-sm font-bold text-wf-heading mb-3`
    - Stacked bar: `flex w-full overflow-hidden rounded-md` with `style={{ height }}`. Each segment is a div with `width: (value/total)*100%`, `backgroundColor: segment.color ?? palette[i % 5]`, and `className="transition-[filter] duration-150 hover:brightness-90"`. Add `title` attribute showing `label: formattedValue` for tooltip.
    - Legend grid (when showLegend=true): `mt-3 grid grid-cols-2 gap-x-4 gap-y-1.5`. Each row has a `rounded-full` colored dot (h-2.5 w-2.5), truncated label (text-xs text-wf-muted), and right-aligned value (text-xs font-medium text-wf-heading tabular-nums ml-auto).
    - Default palette: `['var(--wf-chart-1)', 'var(--wf-chart-2)', 'var(--wf-chart-3)', 'var(--wf-chart-4)', 'var(--wf-chart-5)']`
    - Guard against zero total: use `Math.max(1, segments.reduce(...))` to avoid division by zero.
    - Default formatValue behavior: show percentage `${((value/total)*100).toFixed(0)}%`

    **Anti-patterns to avoid:**
    - Do NOT use Recharts — this is a pure HTML/CSS component
    - Do NOT use Tailwind classes like `bg-wf-chart-1` — they don't exist. Use `style={{ backgroundColor }}` with var() values.
    - Do NOT add BlueprintSection type or SectionRenderer case — this is gallery-only per v1.4 scope.
    - Do NOT use `any` type — strict TypeScript.

    Export as default export.
