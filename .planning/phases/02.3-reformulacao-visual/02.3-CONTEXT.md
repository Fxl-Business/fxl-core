<decisions>
## Implementation Decisions

### Color palette & identity
- New palette entirely — NOT keeping fxl-navy (#1B3A5C) as primary
- Supabase docs as visual inspiration for colors and layout structure
- Both light and dark modes — light as default (business users), dark mode toggle available
- Gold accent color — Claude picks the best tone that feels premium and works in both modes
- Cool grays (slate/zinc) for neutrals — standardize the inconsistent slate usage
- Full CSS variable theme system for light/dark mode switching

### Spacing & density
- Claude's discretion using design skills (frontend-design, ui-ux-pro-max, web-design-guidelines)
- Current app uses text-xs (12px) everywhere — Claude determines optimal typography scale
- Current layout uses max-w-4xl with px-5/py-8 — Claude determines if more breathing room is needed
- Reference: Supabase docs spacing patterns as baseline inspiration

### Design personality
- Claude's discretion using design skills
- No specific product reference beyond Supabase docs for color/layout
- Must feel production-grade — not generic or AI-generated looking
- Internal operational platform — balance professionalism with usability

### Page-level polish
- Claude decides scope, order, and coverage strategy
- Home page: Claude identifies stale content during visual pass and fixes while restyling (e.g. "Referencias" badge should be "Padroes")
- Dark mode toggle placement: Claude decides (TopNav or sidebar footer)
- Sidebar: Claude decides redesign level (full redesign vs restyle) using design skills

### Claude's Discretion
All visual implementation decisions are delegated to Claude with mandatory use of these skills before any visual changes:
- frontend-design — interfaces production-grade com alto design quality
- shadcn — gerenciamento e uso correto de componentes shadcn/ui
- react-best-practices — performance patterns React
- composition-patterns — compound components, render props, context providers
- ui-ux-pro-max — decisoes de design (paletas, tipografia, estilos)
- web-design-guidelines — audit de acessibilidade e boas praticas de UI

Specific areas of full discretion:
- Gold accent exact tone (warm amber, champagne, bright golden — whatever works best)
- Typography scale and hierarchy
- Spacing and layout density
- Sidebar visual evolution level
- Design personality (developer docs vs SaaS vs editorial)
- Page coverage order and priority
- Dark mode toggle placement
- Component polish level (shadows, borders, hover states, transitions)
