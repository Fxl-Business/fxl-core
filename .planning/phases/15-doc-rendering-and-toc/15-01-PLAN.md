   - Update the `pre` component override to use the terminal dots wrapper pattern:
     ```tsx
     pre({ children }) {
       return (
         <div className="group relative my-6 overflow-hidden rounded-xl bg-slate-900 shadow-lg dark:bg-[hsl(var(--code-bg))]">
           <div className="flex items-center gap-1.5 border-b border-slate-700/50 px-4 py-3">
             <div className="h-3 w-3 rounded-full bg-red-500/80" />
             <div className="h-3 w-3 rounded-full bg-yellow-500/80" />
             <div className="h-3 w-3 rounded-full bg-green-500/80" />
           </div>
           <pre className="overflow-x-auto p-4 text-sm leading-relaxed">
             {children}
           </pre>
         </div>
       )
     }
     ```
     Key: The outer div owns the background and rounded corners (overflow-hidden clips). The inner pre has overflow-x-auto for horizontal scroll of long lines. The terminal dots are NOT inside the scrollable pre.

   - Update the `code` component override for inline code:
     Change inline code className from `"rounded bg-muted px-1.5 py-0.5 text-[0.85em] text-primary"` to `"rounded bg-slate-100 px-1.5 py-0.5 text-[0.85em] font-medium text-indigo-600 dark:bg-slate-800 dark:text-indigo-400"`
