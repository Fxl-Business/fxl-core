  <action>
Restyle DocTableOfContents.tsx. The IntersectionObserver logic and heading mapping logic stay exactly the same -- only CSS classes change.

1. Update the outer `aside` element:
   - Change from `"hidden w-52 flex-shrink-0 xl:block"` to `"hidden w-56 flex-shrink-0 xl:block"`
   - Width increase from w-52 (208px) to w-56 (224px) gives more room for heading text.

2. Update the sticky container div:
   - Change from `"sticky top-8 max-h-[calc(100vh-4rem)] overflow-y-auto"` to `"sticky top-24 max-h-[calc(100vh-7rem)] overflow-y-auto"`
   - `top-24` (6rem = 96px) accounts for the 64px header + 32px buffer. This prevents the TOC from overlapping the header.
   - `max-h-[calc(100vh-7rem)]` ensures the TOC can scroll independently if there are many headings.

3. Update the "Nesta pagina" heading:
   - Change from `"mb-3 text-[10px] font-semibold uppercase tracking-widest text-muted-foreground"` to `"mb-3 text-xs font-semibold uppercase tracking-widest text-slate-900 dark:text-foreground"`
   - Change from `text-[10px]` to `text-xs` for slightly larger heading.
   - Change from `text-muted-foreground` to explicit `text-slate-900` for stronger visual presence (matching sidebar section headers from Phase 14).

4. Update the nav element:
   - Change from `"space-y-1"` to `"space-y-1 border-l border-slate-200 dark:border-slate-700"`
   - This adds the border-l rail container that the active indicator overlaps.

5. Update each heading link (`a` element):
   - Change the base className from:
     ```
     'block text-xs transition-colors hover:text-foreground',
     h.level === 3 && 'pl-3',
     activeId === h.id ? 'font-medium text-primary' : 'text-muted-foreground',
     ```
   - To:
     ```
     'block py-1 text-xs transition-colors',
     h.level === 3 ? 'pl-6' : 'pl-4',
     activeId === h.id
       ? '-ml-px border-l-2 border-indigo-600 font-medium text-indigo-600 dark:border-indigo-400 dark:text-indigo-400'
       : 'text-slate-500 hover:text-slate-900 dark:text-slate-400 dark:hover:text-slate-200',
     ```
   - Key changes:
     - Added `py-1` for vertical padding on each link
     - Changed indentation: h2 gets `pl-4`, h3 gets `pl-6` (was: h2 no padding, h3 `pl-3`)
     - Active state uses the Phase 14 `-ml-px border-l-2 border-indigo-600` overlap trick
     - Inactive uses explicit `text-slate-500` instead of `text-muted-foreground`
     - Hover uses `hover:text-slate-900` instead of `hover:text-foreground`
  </action>
