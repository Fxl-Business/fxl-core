# Biblioteca de KPIs — Referência Operacional

> Este documento vive em `docs/referencias/`.
> Referência de KPIs por módulo para apoiar a Fase 1 e reduzir lacunas de escopo.

---

## Regra de uso

- Usar como apoio, não como checklist fechado
- Confirmar coluna de dados antes de considerar um KPI como aprovado
- Sinalizar KPIs sem base de dados como bloqueados

---

## Módulos base

### Comercial
- Faturamento
- Ticket médio
- Conversão
- Vendas por vendedor
- Performance por produto
- Performance por região

### Operacional
- Lead time
- Volume por etapa
- SLA
- Produtividade por equipe
- Taxa de retrabalho

### Financeiro
- Receita
- Despesa
- Margem bruta
- Inadimplência
- Fluxo de caixa
- DRE simplificado

### Inputs
- Volume importado
- Taxa de erro
- Tempo até atualização
- Cobertura de período

---

## Critério de conclusão

O uso desta biblioteca está correto quando o Briefing diferencia:
- KPIs confirmados (coluna de dados validada)
- KPIs sugeridos (aguardando validação)
- KPIs bloqueados (desejados, sem dado disponível)
