# Terraform

## Status no Tech Radar

⬜ **TBD** — IaC para provisionamento de infraestrutura. Decisao pendente.

## O que e

Ferramenta de Infrastructure as Code (IaC) para provisionar e gerenciar infraestrutura de forma declarativa e versionada.

## Quando usar

- Provisionamento de infraestrutura em cloud (AWS, GCP, Azure)
- Quando a infraestrutura precisa ser reproduzivel e versionada
- Ambientes multi-cloud ou com muitos recursos

## Quando NAO usar

- Projetos que usam apenas Vercel + Supabase — a infraestrutura ja e gerenciada
- Escala pequena onde configuracao manual e suficiente
