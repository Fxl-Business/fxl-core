# Docker

## Status no Tech Radar

🔵 **Em avaliacao** — Para containerizacao de backends customizados. Nao aplicavel para frontends na Vercel.

## O que e

Plataforma de containerizacao que empacota aplicacoes e suas dependencias em containers portateis.

## Quando usar

- Backends customizados (Node.js, Python) que precisam de ambiente isolado
- Servicos que rodam fora da Vercel
- Ambientes de desenvolvimento reproduziveis

## Quando NAO usar

- Frontends deployados na Vercel — o deploy ja e gerenciado
- Projetos que usam apenas Supabase + Vercel
