# GitHub Actions

## Status no Tech Radar

🔵 **Em avaliacao** — Em avaliacao para pipelines de lint, test e deploy automatizado.

## O que e

Plataforma de CI/CD integrada ao GitHub. Permite definir workflows automatizados para build, teste e deploy.

## Quando usar

- Pipelines de lint e type-check automatizados em PRs
- Deploy automatico para staging/preview
- Testes automatizados antes de merge

## Quando NAO usar

- Deploys manuais simples (Vercel ja faz preview automatico por PR)
- Workflows muito complexos que justifiquem ferramentas dedicadas
