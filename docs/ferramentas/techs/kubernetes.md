# Kubernetes

## Status no Tech Radar

⬜ **TBD** — Orquestracao de containers. Relevante apenas em escala. Decisao pendente.

## O que e

Plataforma de orquestracao de containers para deploy, escalonamento e gerenciamento automatizado de aplicacoes containerizadas.

## Quando usar

- Multiplos servicos containerizados que precisam de orquestracao
- Escala horizontal automatica
- Ambientes complexos com muitos microservicos

## Quando NAO usar

- Projetos pequenos ou medios — Docker Compose ou deploy direto na Vercel resolve
- Quando o overhead de gerenciar um cluster nao se justifica
