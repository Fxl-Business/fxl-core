## Task 1: Update sync-up.ts for scope, sort_order, and service role key

**Files:**
- Modify: `tools/sync/sync-up.ts`

**Context:** sync-up currently uses the anon key (`VITE_SUPABASE_PUBLISHABLE_KEY`), which is subject to RLS. Product-scoped docs (scope='product') will be rejected by RLS INSERT policies since the anon key has no JWT with super_admin claim. Fix: use the service role key (bypasses RLS) — consistent with how edge functions already access Supabase.

- [ ] **Step 1: Switch to service role key**

In `tools/sync/sync-up.ts`, update the Supabase client setup at lines 22-33. The service role key is needed because product-scoped docs require RLS bypass:

```typescript
const supabaseUrl = process.env.VITE_SUPABASE_URL
const supabaseServiceKey = process.env.SUPABASE_SERVICE_ROLE_KEY

if (!supabaseUrl || !supabaseServiceKey) {
  console.error(
    'Missing environment variables. Ensure VITE_SUPABASE_URL and SUPABASE_SERVICE_ROLE_KEY are set.\n' +
    'Run with: npx tsx --env-file .env.local tools/sync/sync-up.ts'
  )
  process.exit(1)
}

const supabase = createClient(supabaseUrl, supabaseServiceKey)
```

Note: the `.env.local` file must contain `SUPABASE_SERVICE_ROLE_KEY`. Verify it exists before proceeding.

**Files:**
- Modify: `src/modules/docs/hooks/useDocsNav.ts`

- [ ] **Step 1: Group product docs by badge**

In `useDocsNav.ts`, replace the flat product docs mapping (lines 203-206) with badge-based grouping:

```typescript
// Group product docs by badge for hierarchical sidebar
const productByBadge = new Map<string, DocumentRow[]>()
for (const doc of productDocs) {
  // Skip index pages from the flat list — they become group parents
  if (doc.slug.endsWith('/index')) continue
  const badge = doc.badge || 'Outros'
  const group = productByBadge.get(badge) ?? []
  group.push(doc)
  productByBadge.set(badge, group)
}

const groupedProduct: NavItem[] = []
for (const [badge, docs] of productByBadge) {
  const sorted = [...docs].sort((a, b) => a.sort_order - b.sort_order)
  // Find the index page for this badge group (e.g., sdk/index for badge "SDK")
  const indexDoc = productDocs.find(
    (d) => d.badge === badge && d.slug.endsWith('/index')
  )
  if (sorted.length === 1 && !indexDoc) {
    // Single doc without index — render flat
    groupedProduct.push({
      label: sorted[0].title,
      href: `/${sorted[0].slug}`,
    })
  } else {
    // Multiple docs or has index — render as group
    groupedProduct.push({
      label: indexDoc?.title ?? badge,
      href: indexDoc ? `/${indexDoc.slug}` : undefined,
      children: sorted.map((d) => ({
        label: d.title,
        href: `/${d.slug}`,
      })),
    })
  }
}
setProductItems(groupedProduct)
```
